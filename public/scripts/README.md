# Vanta.js Scripts

Place the following Vanta.js script files in this directory:

1. `three.r134.min.js` - Three.js library (version r134)
2. `vanta.fog.min.js` - Vanta.js Fog effect

## Download Instructions

You can download these files from:

1. **Three.js r134**: https://cdn.jsdelivr.net/npm/three@0.134.0/build/three.min.js
   - Save as `three.r134.min.js`

2. **Vanta Fog**: https://cdn.jsdelivr.net/npm/vanta@latest/dist/vanta.fog.min.js
   - Save as `vanta.fog.min.js`

Or download directly from the Vanta.js website: https://www.vantajs.com/

## Quick Download (macOS/Linux)

```bash
cd public/scripts
curl -o three.r134.min.js https://cdn.jsdelivr.net/npm/three@0.134.0/build/three.min.js
curl -o vanta.fog.min.js https://cdn.jsdelivr.net/npm/vanta@latest/dist/vanta.fog.min.js
```
